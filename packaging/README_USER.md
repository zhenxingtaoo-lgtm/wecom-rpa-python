# 企业微信批量转发 RPA 用户说明

1. 打开并登录企业微信。
2. 手动勾选待转发的源消息。
3. 双击 `run-gui.bat`。
4. 在 GUI 中填写本次发送数量。
5. 选择运行模式并完成真实发送确认。
6. 点击“检查环境”，确认源消息和“逐条转发”按钮识别成功。
7. 点击“启动运行”；真实发送模式还需要输入 `SEND`。

程序不使用群列表 CSV，也不保存发送状态数据库。每次重新运行都会从会话
列表底部重新开始，直到达到填写数量或命中哨兵会话。

运行产物：

- `logs/wecom_rpa.log`
- `logs/run_snapshots/*.json`
- `screenshots/`
